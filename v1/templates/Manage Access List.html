How does this work?

If this option is checked, when an Access list item has been used, it will be removed from this list. This will help you keep your Access lists clean and easily see who has not yet taken their Test.

You can always view used Access list items by Exporting Test Results, which will include the Access list items used.

Note:
  • If you allow 3 attempts for the Test, Access list items will be removed only after being used 3 times.
  • If using this Access list across multiple Links, do not use this 'Remove item' option because taking the Test via 'Link A', will then remove the Access list Item from the list which 'Link B' is also using. Instead you could create a separate Access list per Link.

How to apply an Access List?

You can apply an Access List on the Settings page when creating (or editing) a Link.

Tips

  • Access Lists allow you to manage exactly who can access your tests taken via Links.
  • ClassMarker will ask your Access Question and only allow those with valid items to take the test.
  • You can re-use this Access List across multiple Test Links.
